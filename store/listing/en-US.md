# Sirat-i Nur Store Listing Draft - en-US

## Short Description

Prayer times, Qibla, Quran audio, duas, dhikr, and verified Islamic tools.

## Full Description

Sirat-i Nur helps Muslims organize daily worship with prayer times, Qibla
direction, Quran reading and audio, duas, dhikr tracking, reminders, and nearby
Islamic places. The app focuses on verified content, clear offline behavior, and
transparent configuration so users know when cloud content is available and when
the app is using local fallback content.

## Review Notes

- Exact alarms are used only for user-enabled prayer reminders and adhan alerts.
- Location is used only for prayer times, Qibla, and nearby places.
- Premium uses the non-consumable product `sirat_nur_premium_lifetime`.
- Cloud AI mode is optional and uses local fallback when no API key is
  configured.
